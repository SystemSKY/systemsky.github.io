ContainerOPS

Sie entwickeln die Images der Container und wir deployen diese auf unseren Servern

Automatisiertes Container-Deployment - CI/CD




Mit dem Managed Container Deployment Service ContainerOPS der {{ SystemSKY }} müssen sich Entwickler nur noch um die eigenen Container-Images kümmern. Ihre Container können nach dem deployen auf unseren Systemen sofort genutzt werden. Um die dazu benötigten Servern kümmert sich die {{ SystemSKY }}.

Durch ein einfaches Pushen der neuen Image-Version von Ihnen in die Registry, erkennt unser Service automatisch das neue Image und deployt dieses auf unseren Servern. Sie können danach Ihre Applikation in der neuesten Version für Ihre Zwecke benutzen. Dabei wird Ihnen auch ein SSL-Zertifikat zur Verfügung gestellt, damit Ihre Besucher eine verschlüsselte Verbindung aufbauen können.

Wir stellen Ihnen dabei das System so zur Verfügung, dass Sie bei Bedarf eine Datenbank kriegen und einen bereits vorkonfigurierten Mailserver, den Ihre Applikation für den Mailversand benutzen kann. Zusätzlich erhalten Sie noch weitere hilfreiche Funktionen und einen SFTP Zugang zum Server.



Folgende Funktionen stehen Ihnen mit dem ContainerOPS zur Verfügung:

Funktion ContainerOPS
Managed-Server ℹ️ Wir übernehmen die Einrichtung und Betreuung des zugrunde liegenden Serversystems. ✅
SSL-Zertifikat ℹ️ Kostenloses SSL Zertifikat von Let's Encrypt. Wird automatisch verlängert. ✅
HTTP/2 ✅
Gzip Komprimierung ✅
Automatische Container-Updates ℹ️ Unser ContainerOPS erkennt automatisiert ein Update Ihrer Images in der Registry und deployt diese auf den Servern. ✅
Geoblocking ℹ️ Beschränken Sie den Zugriff auf Ihre Dienste auf ausgewählte Länder. ✅
Schutz vor Zugriffen aus dem TOR Netzwerk ✅
Bereitstellung einer Datenbank ✅
Backups ℹ️ Ihre Daten werden täglich gesichert und Ihnen als Dump (auch verschlüsselt möglich) bereitgestellt. ✅
SFTP Zugang ℹ️ Mit dem SFTP können Sie Ihre Dateien auf dem Server verwalten. ✅
ContainerOPS Up-to-date ℹ️ Sie erhalten alle zukünftigen Verbesserungen und alle neuen Features vom ContainerOPS dazu. ✅
Integrierter Mailserver ℹ️ Ein vorkonfigurierter Mailserver mit dem Ihr System Mails versenden kann. ✅
