ServerDays

Wir übernehmen das automatisierte Update-Management Ihrer Linux- und Windowssysteme.

ServerDays




Wenn Sie mehrere Server betreiben, müssen diese regelmäßig aktualisiert werden, um stets auf dem neuesten Stand zu bleiben. Durch Updates werden Fehler in den verwendeten Programmen behoben und Sicherheitslücken geschlossen, die von Hackern ausgenutzt werden könnten.

Wir binden Ihre Server an unser Patch-Management-System an und sorgen für eine zuverlässige und automatisierte Aktualisierung. Den Zeitpunkt der regelmäßigen OS Updates und die Servergruppen können Sie mit uns definieren.

Unser Patch-Management wird dann immer zu diesem Zeitpunkt Ihre Server updaten und automatisiert rebooten. Bei Bedarf können Updates auch kurzfristig eingespielt werden. Unser Service ist sowohl für Linux- als auch für Windows-Server ausgelegt.
